    Lite QQ Extension for VS Code

</div>

> 本扩展基于 **vscode-qq** 扩展开发，由于原作者似乎许久未更新，但是其中的核心库 **oicq** 已更新到2.X版本，有许多弃用和改写的内容，因此本扩展旨在更新原扩展接口，同时在其基础上提升日常使用体验。
> Note: **oicq**库的维护已经失效，核心库更换到维护更活跃的**icqq**库

## Features
- 本扩展不在本地保存任何消息记录和图片
- 支持好友和群聊的分组显示
- 支持显示登录后收到新消息的联系人（重新登录后清空）
- 支持多账号快捷切换
